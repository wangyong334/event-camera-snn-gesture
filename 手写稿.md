## 一、背景与选题（动机）

事件相机（DVS）输出异步事件流 (x, y, t, p)：像素亮度变化时才在精确时刻产生事件，
具有高时间分辨率、低冗余的特点。脉冲神经网络（SNN）以离散脉冲与膜电位时间积分的方式
工作，天然契合事件数据的稀疏、时序特性，并能在神经形态硬件上低功耗运行。手势是随时间
展开的动作，因此本作业选择「事件相机 + SNN」来做手势识别，与本人研究方向（事件相机的
脉冲神经网络）一致。

## 二、数据集

DVS128 Gesture（IBM）：DVS128 相机录制的 11 类手势，29 位受试者、3 种光照，
官方划分 1176 训练 / 288 测试（按人划分）。预处理：把事件流沿时间切成 T=16 个时间片，
每片按极性累积成 2 通道帧，得到输入张量 [T, 2, 128, 128]，T 即 SNN 仿真步数。

## 三、算法架构

【手绘】画一条从左到右的流程：

```
[T,2,128,128]
 → (Conv3×3-BN-PLIF-MaxPool) ×5    边长 128→64→32→16→8→4
 → Flatten → FC → PLIF → FC → PLIF
 → 时间维平均(发放率) → 11类
```

- 神经元用 PLIF（Parametric LIF），膜时间常数 τ 为可学习参数；
- 发放函数不可导，反传用 ATan 替代梯度（surrogate gradient）近似。

## 四、算法步骤

1. 事件编码：异步事件 → [T,2,128,128] 帧；
2. 前向（多步）：输入沿 T 步推进，PLIF 神经元逐步积分膜电位、过阈值发放脉冲；
3. 读出：输出层 T 步脉冲取平均得发放率，作为各类得分；
4. 损失与反传：交叉熵 + 时空反向传播（BPTT），发放处用替代梯度；Adam + 余弦退火 + 混合精度；
5. 复位：每个 batch 后复位膜电位，防止状态跨样本泄漏。

## 五、创新点

1. **PLIF 可学习时间常数**：各层 τ 自适应学习；实验中 τ 随网络加深单调减小
   （2.0 → 1.19），说明浅层「长时记忆」、深层「快速响应」，固定 LIF 无法体现。
2. **稀疏性 / 能耗定量分析**：统计各层发放率，整体稀疏度 88.07%；按 45nm CMOS 简化模型
   （MAC≈4.6pJ、AC≈0.9pJ）估算，SNN 每连接每步能耗约为同规模 ANN 的 1/43，
   定量印证事件相机 + SNN 的低功耗优势。

## 六、实验结果

| 指标 | 数值 |
|------|------|
| 测试准确率 | 78.82% |
| 各层 τ | 1.97 / 2.01 / 1.95 / 1.61 / 1.48 / 1.59 / 1.19 |
| 整体平均发放率 | 0.1193（稀疏度 88.07%） |
| 能耗比 SNN/ANN | 0.0233（≈ 1/43） |

（配置：channels=64, T=16, epochs=64；单卡 RTX 5060 8GB。增大通道数/轮数可进一步提升。）

## 七、结论

设计并实现了用于事件相机手势识别的 PLIF 脉冲卷积网络，在 DVS128 Gesture 上达到 78.82%
准确率。各层学到的时间常数随深度递减，且网络保持约 88% 的发放稀疏度、能耗约为 ANN 的
1/43，验证了「事件相机 + SNN」在时序建模与低功耗方面的优势。
